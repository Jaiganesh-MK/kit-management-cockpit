# Mandatory Safety Kits — Curing Line Series

## Overview

Mandatory safety kits (MK_Safety) are non-negotiable regulatory retrofits. Non-installation by the stated deadline results in loss of CE certification and Tetra Pak warranty void. These kits must be installed regardless of customer consent.

## Safety Guard Retrofit Kit — 3495770-02002

### Applicable Equipment
- Material: 3000-MAT-CUR-150, 3000-MAT-CUR-200
- Jaw type: Standard jaw only (not Enhanced)
- Cap size: 18mm and 22mm

### Safety Directive
Directive SD-2024-447 mandates installation on all Curing Line 150ml and 200ml machines due to identified pinch hazard at jaw closure position 3. Issued by Tetra Pak Global Safety Board, 2024-03-15.

### Deadline
31-December-2025. Machines not retrofitted by this date cannot be operated under Tetra Pak service agreement.

### Prerequisites
- Base rail kit 3495770-01003 must be installed before this kit
- Machine must be powered down with lockout-tagout applied
- Jaw must be at home position before installation begins

### Installation Procedure Summary
1. Verify base rail kit is installed and torqued to 45 Nm
2. Remove existing jaw guard (2 M8 bolts)
3. Fit new safety guard bracket — align to reference marks A and B
4. Secure with 4 M8 bolts, torque to 45 Nm
5. Fit guard panel, connect sensor harness connector J3
6. Power on and verify sensor signal in HMI — must show GREEN
7. Run 10 dry cycles to confirm no interference

### Skills Required
- Certified Tetra Pak mechanical technician
- Safety certification SC-03 or above

### Estimated Downtime
4 hours off-shift. Cannot be done during production.

### Known Issues
- If sensor shows RED after installation, re-check harness connector J3 seating
- Older base rail kits installed before 2020 may need shim kit 3495770-00001 — check part date code


## MK Corrective — Drive Belt Tensioner Kit — 3495880-01002

### Applicable Equipment
- Material: 3000-MAT-CUR-150
- All jaw configurations

### Issue Description
Drive belt tension loss identified in field after 60,000 operating cycles. Symptom: intermittent jaw timing errors, package weight variation.

### Deadline
No hard deadline. Install at next planned service if fault code TE-04 or TE-07 is present.

### Installation Procedure Summary
1. Remove drive cover (6 M6 bolts)
2. Release belt tension using tensioner tool TP-880
3. Remove old belt — check pulley for wear, replace if diameter below 48.5mm
4. Fit new belt, set tension to 280N using TP-880
5. Refit cover, run 5 production cycles and verify belt tension unchanged

### Skills Required
- Mechanical technician
- No special certification required

### Estimated Downtime
6 hours including machine cooldown
